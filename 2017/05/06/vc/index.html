项目总结：掌握了vue.js、react的用法，和他们之间的一些区别和特性，也遇到了诸如一下几类问题：
react：

  1. react中注释只能用双斜杠
  2. render渲染页面的时候必须要有一个根节点
  3. 给标签加class的时候只能用className
  4. 循环数据的时候必须要有一个独一无二的key

vue.js:

  1. 当移动文件夹时会出现报错的情况，需要重新下载node modules里面的插件
  2. css样式格式不正确时也会报错
